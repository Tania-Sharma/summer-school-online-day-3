About My Internship

I am really excited to share that I got selected for a summer internship with IIT Jammu in collaboration with Techible. I applied through the official Summer School website. There were many domains like Python, AI, and IoT, but I chose Web Development because I wanted to build my basics strong.

Honestly, I didn’t know much before this, but now I feel confident to learn step by step. I believe starting with basics is really important for any beginner.

Why Web Development?

Since long, I have been curious about how websites work and look so clean and interactive. I wanted to learn how everything shows up when we click buttons or fill forms.

I feel web development is beginner-friendly too. I can see results quickly on my screen — which keeps me motivated. Someday, I want to build my own personal website completely by myself!

Learning So Far

We started with HTML basics — how to write headings, paragraphs, links, images. It felt so good to see my name and my first web page open in the browser! After that, we did CSS — styling, colors, spacing, layouts. Now, we have started with JavaScript and learned how to add simple interactivity. Our mentor, Nitin Raina Sir, explains everything in an easy way.

Hands-On Tasks

After every class, we get small assignments. My first task was to make a simple portfolio page using HTML. The second was designing a profile card with HTML & CSS. The third one is this blog — where I’m using CSS Grid and sticky layout for the first time!

Doing these tasks makes me more confident. I also Google solutions and watch short videos when I get stuck — and that’s helping me learn faster.

Orientation & Extra Sessions

Before the classes began, we had an orientation session where mentors explained how everything will work. We were also guided on writing LinkedIn posts and why it’s important to share our progress.

We even got an extra module about AI tools — which was something new for me. It opened my mind to how AI can help students and developers both!

Sunday Masterclasses

Every Sunday, there is a special masterclass with an industry expert. So far, we have learnt about how to build a strong resume, tips for interviews, and why DSA (Data Structures and Algorithms) is important. These sessions are very motivating and give us an idea about real industry expectations.

What’s Next?

I am excited for the upcoming classes and new projects. There is still a lot more to learn — React, Node, MongoDB — all are waiting for us! I hope to make the most of this internship and build a strong base for my future projects.
